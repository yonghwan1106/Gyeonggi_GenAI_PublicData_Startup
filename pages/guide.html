계정 생성

상권 마스터를 시작하려면 먼저 계정을 생성해야 합니다.

  1. 메인 페이지에서 '무료 체험' 버튼을 클릭합니다
  2. 이름, 이메일, 전화번호를 입력합니다
  3. 업종을 선택합니다
  4. 이용약관에 동의하고 가입을 완료합니다
14일 무료 체험 기간 동안 모든 기능을 제한 없이 사용할 수 있습니다.
업체 정보 등록

정확한 분석을 위해 업체 정보를 정확히 입력해주세요.

  • 업체명: 정확한 상호명을 입력합니다
  • 주소: 도로명 주소를 정확히 입력합니다
  • 업종: 주력 업종을 선택합니다
  • 영업시간: 평일/주말 영업시간을 설정합니다

AI가 당신의 브랜드에 맞는 SNS 콘텐츠를 자동으로 생성합니다.

단계별 가이드
  1. AI 마케팅 메뉴로 이동
  2. SNS 포스트 탭 선택
  3. 업체명, 업종, 주요 서비스 입력
  4. 포스트 목적 선택 (신메뉴 홍보, 할인 이벤트 등)
  5. 'AI 콘텐츠 생성' 버튼 클릭
  6. 생성된 콘텐츠 확인 및 수정
  7. 복사하여 SNS에 게시
